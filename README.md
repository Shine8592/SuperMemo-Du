# 嘟嘟超级记忆系统 (SuperMemo-Du)

> 一个自主进化的AI记忆系统，具备自动双记忆协同引擎、三级存储管理器和单命令恢复系统

## 🚀 核心功能

### 🔄 自动双记忆协同引擎 (dual-memory-core)
- **短期记忆**：实时捕获会话上下文，24小时内保持高保真
- **长期记忆**：通过深度提炼和关联建立永久知识库
- **智能协同**：两种记忆形式动态平衡，确保信息不丢失且不冗余

### 💾 三级存储管理器 (storage-manager)
1. **热存储 (Hot)**：当前会话活跃数据，毫秒级访问
2. **温存储 (Warm)**：最近重要记忆，快速检索  
3. **冷存储 (Cold)**：长期永久存档，成本最优

### ⚡ 单命令恢复系统 (one-click-recovery)
- 一键恢复到任意历史状态
- 自动冲突检测和智能合并
- 零配置快速回滚
